**📌 ENVIO DE EMAILS (SIMULAÇÃO)**
Descrição do Exercício:

Neste exercício, trabalhei com manipulação de arrays em JavaScript, utilizando os métodos map e forEach. A partir de uma lista de e-mails, criei uma nova estrutura contendo objetos com e-mail e nota, e simulei o envio de mensagens para cada item da lista.

**Estrutura:**
Lista de dados:

Criei um array contendo e-mails de pessoas aprovadas.

**Transformação dos dados:**

Utilizei o método map para transformar a lista de e-mails em uma lista de objetos, adicionando a propriedade nota para cada item.

**Execução da ação:**

Utilizei o método forEach para percorrer a lista de objetos e executar uma função responsável por simular o envio de e-mails.

**Função utilizada:**

Criei a função enviarEmail, que recebe um objeto como parâmetro e exibe uma mensagem no console simulando o envio do e-mail.


**Observação:**

Inicialmente, identifiquei um erro na interpolação de variáveis devido ao uso de aspas simples, corrigido com a utilização de crases (`), permitindo a exibição correta dos dados.
